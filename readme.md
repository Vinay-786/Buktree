# FOLLOW THE BELOW COMMANDS TO SET UP THE PROJECT:

* create an environment using:
    ```bash
    python3 -m venv env

* activate the virtual environment:
    ```bash
    source env/bin/activate

* Install the requirements:
    ```bash
    pip install -r requirements.txt

# TO RUN THE PROJECT:

* python run.py
